Football Transfer Market Analysis

This project explores transfer market efficiency in professional football using publicly available transfer data.

The analysis focuses on identifying whether top football clubs tend to overpay for players, or whether they operate more efficiently compared to the rest of the market.

Key Ideas

The main hypothesis explored in this notebook:

Top football clubs do not necessarily overpay for players, but may operate more efficiently relative to player market value.

To evaluate this, the notebook introduces a custom metric:

price_ratio = transfer_fee / market_value

This allows transfer fees to be normalized against estimated player market values.
